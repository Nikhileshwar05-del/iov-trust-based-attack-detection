#  Blockchain-Based Intelligent Attack Detection in IoV

##  Project Overview

This project presents a secure and intelligent system for detecting malicious vehicle behavior in Internet of Vehicles (IoV) using Blockchain and Machine Learning.

##  Modules

### 1. IoV Simulation

* SUMO-based vehicle simulation
* Generates realistic traffic and attack data

### 2. Blockchain

* Secure logging using Proof of Work
* RSU-based decentralized system

### 3. Machine Learning

* Random Forest classifier
* Multi-class classification:

  * Normal
  * Suspicious
  * Critical
  * Malicious

##  Attack Types

* DoS (Denial of Service)
* Sybil Attack
* Spoofing
* False Data Injection

## Features Used

* msgRate, delay, drop, fmr
* speed, acceleration
* neighbors, distance

## Dataset

* iov1_dataset.csv
* blockchain1_log.csv

##  How to Run

1. Run simulation to generate dataset
2. Run blockchain logging script
3. Run ML model

##  Results

* High accuracy in classification
* Effective detection of malicious vehicles

* Python
